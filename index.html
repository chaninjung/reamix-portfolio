Features
6가지 핵심 기능
오디오 믹싱의 전체 워크플로우를 커버하는 AI 파이프라인.
🎛️

자연어 믹싱

"리버브 걸어줘", "0:05~0:15 구간만 밝게" 같은 자연어를 333개 믹싱 컨셉에 매칭하여 REAPER 파라미터로 변환합니다.

Claude / Gemini Embedding Cosine Similarity
👤

유저별 개인화

유저의 과거 선택과 피드백을 학습하여, 같은 요청도 유저마다 다른 파라미터로 해석합니다. EMA 기반 선호도 추적.

STM/LTM Memory EMA Tracking Feedback Loop
🎵

AI 스템 분리 (Demucs)

HTDemucs 모델로 보컬, 드럼, 베이스, 기타를 AI 기반 분리. 재귀 분리로 "other" 스템을 다시 세분화합니다.

Demucs Recursive Async Job
🔊

Mid/Side 스테레오 분리

순수 DSP로 스테레오 신호를 센터(Mid)와 사이드(Side)로 분리. torchaudio 기반 실시간 처리.

torchaudio DSP
🔬

Gemini 오디오 분석

Gemini 멀티모달 API로 오디오 파일을 직접 분석하여 악기 식별, 트랙 이름 제안, EQ 프로필을 추천합니다.

Multimodal gemini-2.5-flash
🔗

임베딩 시맨틱 매칭

333개 믹싱 컨셉을 벡터화하여 코사인 유사도로 매칭합니다. DB에 없는 요청은 LLM이 직접 EQ 파라미터를 생성(raw_eq fallback)합니다.

Embedding Vector Cache LLM Fallback
