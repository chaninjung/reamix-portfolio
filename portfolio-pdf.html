  • BM25 + Kiwi 형태소 분석 — 건설 전문 용어 최적화
  • K=5 전문 로드 — 청크가 아닌 문서 전체를 LLM에 전달
  • 비용: Agent 4회(33원) → RAG 1회(~10원) 3배 절감

LoRA 파인튜닝 경험

  • CAD LISP 코드 생성 모델 연구
  • LLM-as-Judge 90점+ 필터링, 3,800개 데이터 구축
  • instruction-answer: 38% → chain-of-thought: 90%

품질 평가 체계

  • 평가 설계 주도: O/X 바이너리 판정 + 출처(reference) 측정 방식 제안
  • 팀원과 함께 50개 검증 데이터셋 구축
  • 출처 검색 성공률 Recall@5: 82.0%
  • 출처 검색 성공 시 정확도: 87.8% vs 실패 시: 66.7%
  • false positive 11건 → 0건 제거

ReaMix AI — 유저별 개인화 엔진

같은 말, 다른 해석 · STM/LTM 메모리 + EMA 선호도 추적

User A — 내추럴 톤

Style: natural/transparent · preferred_intensity: 0.30

"로우미드 살려서 바디감 줘"

ReaEQ: 280Hz +1.5dB (Q 0.8)
Low Shelf: 120Hz +1.0dB
→ 절제된 바디감

User B — 아날로그 톤

Style: analog/colored · preferred_intensity: 0.90

"로우미드 살려서 바디감 줘"

ReaEQ: 250Hz +4.5dB (Q 0.6)
Low Shelf: 120Hz +3.0dB
HF Rolloff: 6kHz −1.5dB
→ 풍성한 아날로그 톤

피드백 → 학습 → 반영

User B: "로우미드가 좀 과한데" → intensity 0.90→0.71 자동 조정 → 다음 요청에서 EQ +3dB로 감소

메모리 아키텍처

STM: 최근 50개 대화 (rolling window)
LTM: 선호도, 스타일 태그, 프로필 노트 (영구 저장 + 자동 요약 압축)
